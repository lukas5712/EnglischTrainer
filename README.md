## ✨ Features

- Multiple training modes:
  - Vocabulary
  - Sentence
  - Translate
  - Multiple Choice
  - Grammar Test
  - Vocab Test
- AI-powered exercises
- Progress tracking (XP, Level, Streak)
- Favorites & mistake training
- Clean modern UI
- One-click setup (automatic Python installation)
